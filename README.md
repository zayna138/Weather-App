# Weather App

## Project Overview
This project is a Python-based weather application that fetches real-time weather data using the OpenWeather API.

## Features
- Real-time weather updates
- Temperature display
- Humidity display
- Wind speed display
- Weather condition display

## Technologies Used
- Python
- Requests Library
- OpenWeather API

## How to Run

1. Install requests library:
pip install requests

2. Run the project:
python main.py
